O que é

Uma caixinha para abrir conversas que a família vai guardar para sempre

Este livro-caixinha é um convite para contar histórias, falar sobre sonhos antigos e futuros, transmitir valores e resgatar lembranças que merecem ser lembradas e compartilhadas.

Há perguntas pensadas só para os avós, perguntas só para os netos e outras para todo mundo responder. A ideia é simples: despertar curiosidade, fortalecer vínculos e criar espaço para conversas que muitas vezes a correria da vida deixa para depois.

FAQ

As dúvidas que importam

É só para crianças pequenas?

Não. A proposta funciona com diferentes idades, desde que exista abertura para conversar e compartilhar.

Precisa seguir ordem?

Não. As perguntas podem ser usadas livremente, de acordo com o momento.

É uma boa opção de presente?

Sim. É um presente afetivo, original e cheio de significado.

Serve para famílias que não convivem todos os dias?

Sim. Inclusive pode funcionar muito bem como ponte para aproximar quem se encontra menos.
